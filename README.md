# 📖 Overview

The Internship Management System is a centralized web platform developed for educational institutions to efficiently manage internship records, applications, evaluations, reports, and communication between Students, Faculty Coordinators, and Administrators.

The portal automates the complete internship workflow while providing a secure and user-friendly interface.

# 🚀 Installation

1. Clone the repository

```

```

2. Import the sample database

3. Configure

```

config.php

```

4. Start Apache & MySQL

5. Open

```

localhost/Internship-Management-System

```
